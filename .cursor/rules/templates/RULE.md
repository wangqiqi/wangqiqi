# 🎨 项目配置模板框架

这个规则包含了项目配置的模板框架，用于自动化生成项目初始化配置，特别是针对文档管理和工作日志项目的专项配置。

## 📋 模板文件

- **`templates.json`** - 完整的项目配置模板，包含：
  - 项目元数据
  - 协作设置
  - 技术要求
  - 部署配置
  - **文档管理配置** (针对文档项目和工作日志项目)

## 🔧 使用方法

当需要生成新的项目配置时：

1. 复制 `templates.json` 作为基础模板
2. 根据项目需求修改相应配置
3. 运行项目初始化脚本应用配置

## 📊 配置覆盖范围

- **协作模式**: 自动/确认式选择
- **质量等级**: 标准/企业级质量控制
- **安全级别**: 不同层级的安全要求
- **部署策略**: 环境分离和回滚能力
- **文档管理**: 文档规范、工作日志模板、协作流程 (针对文档项目)

## ⚙️ 自定义配置

可以通过修改 `templates.json` 中的条件逻辑来自定义配置生成规则：

```json
{
  "collaboration_settings": {
    "mode": "{{TEAM_SIZE == 'personal' ? 'autonomous' : 'confirmatory'}}"
  }
}
```
